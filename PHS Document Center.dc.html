ไฟล์ทั้ง 3 นี้คือชุดข้อมูลหลักที่ AI ต้องอ่านก่อนเริ่มทำงาน — ดาวน์โหลดให้ครบก่อนไปขั้นตอนถัดไป

4. เปิด HANDOVER PROMPT
เปิดไฟล์แล้วคัดลอกข้อความทั้งหมดตั้งแต่บรรทัดแรกจนถึงบรรทัดสุดท้าย
📋 เปิด HANDOVER PROMPT
5. วาง Prompt ใน ChatGPT
วางข้อความ HANDOVER PROMPT ในข้อความเดียวกับที่แนบไฟล์ทั้ง 3 ไว้ (ไม่ใช่ส่งแยกเป็นสองข้อความ) → ตรวจสอบว่าไฟล์ทั้ง 3 ยังแสดงอยู่ → กดส่งครั้งเดียว
6. รอ AI อ่านและสรุป
AI ควรอ่าน START HERE ก่อน → อ่าน Mission Control เพื่อดูสถานะปัจจุบัน → อ่าน BLUEPRINT เพื่อเข้าใจโครงการ → สรุปความเข้าใจ → ระบุ Current Phase → รอคำสั่งจาก CEO

9. หากไม่ผ่าน
อย่าเริ่มทำงานต่อ → ตรวจสอบว่าแนบไฟล์ครบหรือไม่ → ตรวจสอบว่าใช้ไฟล์เวอร์ชันล่าสุดหรือไม่ → ส่ง HANDOVER PROMPT ใหม่อีกครั้ง → หากยังไม่ผ่าน ให้กลับไปตรวจ Mission Control และ BLUEPRINT
