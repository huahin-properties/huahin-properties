# huahin.properties — อัปเดตล่าสุด (รูปเจ้าของ + จัดเรียง FB ใหม่ + ระยะทางสถานที่ใกล้เคียงอัตโนมัติ)

## สรุปสิ่งที่แก้ในรอบนี้
1. **รูปเจ้าของทรัพย์** — หน้ากรอกข้อมูลเจ้าของ (ขั้นตอนก่อนเผยแพร่จริง) มีช่องอัปโหลดรูป 2 รูปแล้ว (เก็บให้แอดมินดูเท่านั้น)
2. **จัดเรียงโพสต์ Facebook ใหม่** — "(ภาษาไทยด้านล่าง)" ย้ายไปต่อจากหัวข้อ SEO (ไม่แย่งตำแหน่งบรรทัดแรกอีกต่อไป) และแฮชแท็กย้ายไปอยู่ด้านล่างสุดใต้ข้อมูลติดต่อ ไม่บังสายตาด้านบนอีกต่อไป
3. **ระยะทางสถานที่ใกล้เคียงอัตโนมัติ (ใหม่)** — เพิ่มช่อง "ลิงก์ Google Maps" ในหน้าฟอร์ม วางลิงก์พิกัดบ้าน ระบบจะค้นหาสถานที่ใกล้เคียงยอดนิยม 10 แห่งรอบๆ พิกัดนั้น (แหล่งท่องเที่ยว ห้าง โรงพยาบาล สนามบิน ฯลฯ) พร้อมคำนวณระยะทาง+เวลาขับรถจริงให้อัตโนมัติ แล้วนำไปเขียนลงในคำบรรยายทรัพย์ (ภาษาไทย) อย่างเป็นธรรมชาติ — ข้อมูลนี้จะติดไปทั้งหน้าเว็บไซต์และโพสต์ Facebook เพราะมาจากคำบรรยายชุดเดียวกัน
   - ใช้ Google Maps API key เดิมที่มีอยู่แล้ว (เปิด Places API + Distance Matrix API เพิ่มแล้วในโปรเจกต์เดียวกัน)
   - ถ้าลิงก์เป็นลิงก์แชร์แบบสั้น (maps.app.goo.gl) ระบบจะแจ้งเตือนให้เปิดลิงก์ในเบราว์เซอร์ก่อนแล้ววาง URL เต็มจาก address bar แทน (มีพิกัดตัวเลขอยู่ในนั้น)

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบ "เพิ่มทรัพย์ด่วนด้วย AI" จากหน้าเว็บจริง — ลองใส่ลิงก์ Google Maps ดูว่าค้นหาสถานที่ใกล้เคียงได้ไหม
