# huahin.properties — อัปเดตล่าสุด (สถานที่ใกล้เคียงครบ 10 แห่ง + จัดเรียงฟอร์มใหม่)

## สรุปสิ่งที่แก้ในรอบนี้
1. **สถานที่ใกล้เคียงครบตามที่ขอแล้ว** — เปลี่ยนจากให้ AI เลือกเขียนเองบางส่วน (ทำให้ได้ไม่ครบ 10 แห่ง และมีแค่ภาษาอังกฤษ) เป็น**ระบบสร้างรายการให้ครบทุกแห่งเองแบบตายตัว** ต่อท้ายคำบรรยาย ทั้งภาษาไทยและอังกฤษ(+รัสเซีย/จีน/เยอรมัน/นอร์เวย์) รูปแบบ:
   ```
   สถานที่ใกล้เคียง:

   - ชื่อสถานที่ ห่างประมาณ 1.5 กม. ใช้เวลาขับรถประมาณ 15 นาที

   - ชื่อสถานที่ 2 ห่างประมาณ 2 กม. ใช้เวลาขับรถประมาณ 10 นาที
   ```
   แต่ละแห่งขึ้นบรรทัดใหม่ เว้นบรรทัดคั่น อ่านง่าย เรียงสวยงามตามที่ขอ
2. **จัดลำดับฟอร์มใหม่** — ข้อ 1 เป็นพิกัดแผนที่แล้ว (ตามด้วยรูปภาพ, เอกสาร, พูดอธิบาย) พร้อมตัวอย่างรูปแบบพิกัดที่ถูกต้อง `12.558940,99.909039` ให้เห็นชัดเจนในหน้าฟอร์ม กันสับสน

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบใส่พิกัดแบบ `12.558940,99.909039` ในข้อ 1 ของหน้า "เพิ่มทรัพย์ด่วนด้วย AI" แล้วดูผลลัพธ์ในคำบรรยาย
