# huahin.properties — อัปเดตล่าสุด (ลำดับขั้นตอนเผยแพร่ + คำบรรยาย Facebook)

## สรุปสิ่งที่แก้ในรอบนี้
1. **คำบรรยายบ้านในโพสต์ Facebook** — เพิ่มข้อกำหนดให้ AI เขียนคำบรรยายทรัพย์แบบเต็ม (ไม่ใช่แค่ข้อมูลดิบ) ทั้งภาษาอังกฤษและไทย เหมือนคำโปรยขายจริง
2. **ย้าย "(ภาษาไทยด้านล่าง)"** ไปไว้บรรทัดบนสุดของโพสต์ พร้อมธงชาติไทย 🇹🇭 ให้ลูกค้าเห็นทันที
3. **ลำดับขั้นตอนเผยแพร่ใหม่** — จากหน้าตรวจสอบร่าง กดต่อไปจะต้อง**กรอกข้อมูลเจ้าของทรัพย์ก่อน** (ชื่อ+เบอร์โทรบังคับ) แล้วจึงกดบันทึกและเผยแพร่จริง จากนั้นข้อความสำหรับคัดลอกไปโพสต์ Facebook จึงปรากฏเป็นขั้นตอนสุดท้าย — เรียงลำดับตามที่ควรเป็น ไม่ต้องย้อนกลับไปหน้า Admin Dashboard อีก

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบ "เพิ่มทรัพย์ด่วนด้วย AI" จากหน้าเว็บจริง (ไม่ใช่หน้าพรีวิวในระบบนี้ — มีข้อจำกัดขนาดไฟล์)
