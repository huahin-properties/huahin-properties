# huahin.properties — อัปเดตล่าสุด (SEO keyword ตามสถานะขาย/เช่า ให้ถูกต้องเสมอ)

## สรุปสิ่งที่แก้ในรอบนี้ (ต่อจากไฟล์ก่อนหน้า)
- ขยายคลังคีย์เวิร์ด SEO ภาษาไทย+อังกฤษให้ครอบคลุมมากขึ้น (ขายบ้านพร้อมอยู่, ขายบ้านใกล้ทะเล, Move-in Ready, Beachfront Property ฯลฯ)
- **บังคับให้คีย์เวิร์ดตรงกับสถานะจริงของทรัพย์เสมอ** — ทรัพย์ที่เป็น "เช่า" จะได้คีย์เวิร์ดเช่า/ให้เช่า/For Rent เท่านั้น ทรัพย์ที่เป็น "ขาย" จะได้คีย์เวิร์ดขาย/For Sale/Buy เท่านั้น ไม่ปนกันอีก — มีผลทั้งชื่อทรัพย์ คำโปรย SEO keywords ที่เก็บในระบบ และแคปชั่น Facebook

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบ "เพิ่มทรัพย์ด่วนด้วย AI" จากหน้าเว็บจริง (ไม่ใช่หน้าพรีวิวในระบบนี้)
