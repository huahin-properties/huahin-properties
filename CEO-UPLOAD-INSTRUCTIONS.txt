คำแนะนำการอัปโหลด — ใช้ไฟล์นี้เท่านั้น

1. นี่คือ ZIP เดียวที่ CEO ควรใช้สำหรับอัปโหลดรอบนี้ ไม่ต้องใช้ไฟล์ดาวน์โหลดเก่าอื่นใด
2. แตกไฟล์ (Unzip) ก่อน
3. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดที่อยู่ข้างในไปยัง GitHub repository เดิมที่ใช้อยู่
4. ให้เขียนทับไฟล์ชื่อเดิมที่มีอยู่แล้ว (Overwrite)
5. ห้ามอัปโหลดไฟล์ ZIP เองเข้าไปใน repository
6. ห้ามลบการตั้งค่า GitHub อื่นที่ไม่เกี่ยวข้อง
7. หลังอัปโหลดเสร็จ ให้รอสักครู่ให้ GitHub Pages อัปเดตเว็บไซต์ แล้วเปิดตรวจสอบหน้าที่จำเป็น (CEO Guide, PHS Document Center, Complete Snapshot, และหน้าเว็บหลัก)
