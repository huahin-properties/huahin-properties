# huahin.properties — เพิ่มธงชาติฝรั่งเศส + อิตาลี

## สิ่งที่แก้ในรอบนี้
เพิ่มไอคอนธงสีในเมนูเปลี่ยนภาษาสำหรับ **Français** (ธงฝรั่งเศส น้ำเงิน-ขาว-แดง) และ **Italiano** (ธงอิตาลี เขียว-ขาว-แดง) — ก่อนหน้านี้ 2 ภาษานี้ยังเป็นสีเทาว่างเพราะยังไม่มีข้อมูลธงในโค้ด

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด — ไฟล์ที่เปลี่ยนจริงคือ `LanguageSwitcher.dc.html`)
2. รอ GitHub Pages deploy เสร็จ แล้ว Hard Refresh (Ctrl+Shift+R)

## ยังไม่ได้แก้ (ค้างจากรอบก่อน)
เรื่องแก้ไข "กรรมสิทธิ์ (Ownership)" ให้แปลได้ครบทุกภาษา — จะทำในรอบถัดไป
