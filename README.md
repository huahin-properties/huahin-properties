# huahin.properties — ไฟล์เว็บล่าสุด (ไมค์ในกล่องแก้ไข + แปลภาษาแม่นยำขึ้น + สุ่มภาษารูป)

## อัปเดตล่าสุด
1. **เพิ่มปุ่มไมค์ในกล่อง "พูด/พิมพ์แก้ไขเพิ่มเติมให้ AI"** — ตอนนี้พูดแก้ไขเพิ่มเติมได้เหมือนกล่องแรก ไม่ต้องพิมพ์เท่านั้นแล้ว
2. **แก้การแปลภาษา** — เดิม AI ถูกบังคับให้มองว่าเสียงพูด/ข้อความที่ป้อนเป็นภาษาไทยเสมอ ตอนนี้แก้ให้ AI เข้าใจและแปลได้ถูกต้องไม่ว่าจะพูดภาษาไทยหรือภาษาอังกฤษ (หรือปนกัน) แล้วสรุปร่างหลักเป็นภาษาไทยเสมอ ก่อนแปลต่อเป็นอีก 5 ภาษา
3. **สุ่มภาษาชื่อไฟล์รูปภาพ 50/50** — เดิมตั้งชื่อรูปเป็นภาษาไทยทั้งหมด ตอนนี้ให้ AI สลับภาษาไทย/อังกฤษแบบสุ่มในแต่ละรูป (คงคีย์เวิร์ด SEO ไว้ในทั้งสองภาษา) เพื่อดึงกลุ่มลูกค้าทั้งคนไทยและต่างชาติ

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบปุ่ม 🤖 "เพิ่มทรัพย์ด่วนด้วย AI" อีกครั้ง
