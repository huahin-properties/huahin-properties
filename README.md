# huahin.properties — เพิ่ม Contact Rail (แถบติดต่อด้านขวา)

## สิ่งที่เพิ่มในรอบนี้
เพิ่มไฟล์ใหม่ `ContactRail.dc.html` — แถบติดต่อลอยด้านขวามือ แสดงบนทุกหน้าสาธารณะ (หน้าแรก, ค้นหา, รายละเอียดทรัพย์, ขาย/ฝากขาย, เกี่ยวกับเรา, ติดต่อ):

- **3 แท็บ**: "ติดต่อเรา" (เปิดฟอร์มกรอกชื่อ/เบอร์/อีเมล/ข้อความ), LINE, WhatsApp — คลิกแล้วเข้าใช้งานได้ทันที
- **ลูกศรเล็กๆ ซ่อน/แสดง** แถบทั้งหมด กดครั้งเดียวซ่อน กดอีกครั้งแสดงกลับมา — จำค่าไว้ (ครั้งหน้าเข้ามาจะเป็นแบบที่ตั้งค่าไว้ล่าสุด)
- **ปุ่ม "แชทกับเรา" มุมล่างขวา แสดงตลอดเวลา** ไม่ว่าจะซ่อนแถบหรือไม่ — กดแล้วเปิดฟอร์มติดต่อเดียวกัน
- รองรับทั้ง 8 ภาษาที่มีในเว็บ (ไทย/อังกฤษ/รัสเซีย/จีน/เยอรมัน/นอร์เวย์/ฝรั่งเศส/อิตาลี) — ทดสอบแล้วภาษาฝรั่งเศสใช้งานได้ครบ
- ปรับ UI ให้เหมาะกับมือถือ (ตำแหน่งขวา ขนาดพอดีนิ้วมือ)

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด — ไฟล์ใหม่ที่ต้องเพิ่มคือ `ContactRail.dc.html`)
2. รอ GitHub Pages deploy เสร็จ แล้ว Hard Refresh (Ctrl+Shift+R)
3. เบอร์โทร/LINE ที่ใช้ตอนนี้เป็นเบอร์เดียวกับหน้า Contact เดิม (+66 32 000 000, line.me/ti/p/huahinproperties) — แจ้งได้ถ้าต้องการเปลี่ยน

## ยังไม่ได้แก้ (ค้างจากรอบก่อน)
เรื่องแก้ไข "กรรมสิทธิ์" ให้แปลได้ครบทุกภาษา — จะทำในรอบถัดไป
