# huahin.properties — ไฟล์เว็บล่าสุด (ฟีเจอร์ AI ร่างประกาศใช้งานได้จริงบนเว็บจริงแล้ว!)

## 🎉 อัปเดตสำคัญที่สุดในชุดนี้
**ฟีเจอร์ "เพิ่มทรัพย์ด่วนด้วย AI" ใช้งานได้จริงบนเว็บจริงแล้ว** — ก่อนหน้านี้ทำงานได้เฉพาะในหน้าต่างพรีวิวของระบบนี้ ตอนนี้เชื่อมต่อกับ Firebase Cloud Function (ตัวกลางที่เก็บ Anthropic API key ไว้อย่างปลอดภัยฝั่งเซิร์ฟเวอร์) เรียบร้อยแล้ว ปุ่ม "ให้ AI ร่างประกาศ" และ "ปรับปรุงร่าง" จะทำงานได้เต็มรูปแบบทันทีที่อัปโหลดไฟล์ชุดนี้ขึ้นเว็บจริง

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. เข้า `Admin Login.dc.html` ล็อกอินด้วย `132435` / `qewret`
4. ทดสอบปุ่ม 🤖 "เพิ่มทรัพย์ด่วนด้วย AI" ในแดชบอร์ด — อัปโหลดรูป/พูดอธิบาย/กด "ให้ AI ร่างประกาศ" ควรทำงานได้เลย

## สรุประบบทั้งหมดที่ใช้งานได้แล้ว
- **Firestore** — เก็บข้อมูลทรัพย์/เจ้าของ/ผู้เช่า/รูปภาพ ให้ทุกคนเห็นตรงกันแบบเรียลไทม์
- **Firebase Cloud Function** — ตัวกลางเรียก Claude AI อย่างปลอดภัย (เก็บ API key ไว้ที่ `functions:secrets`)
- **ระบบล็อกอิน Admin** — `132435` / `qewret` (เปลี่ยนได้ที่หน้า Site Content)
- **อัปโหลดรูปภาพ** — เก็บใน Firestore โดยตรง (บีบอัดอัตโนมัติ)
- **AI Quick Add** — อัปโหลดรูป+เอกสาร+เสียงพูด → AI ร่างประกาศ 6 ภาษา → ตรวจสอบ → เผยแพร่

## ⚠️ ข้อควรรู้ก่อนใช้งานจริงเต็มรูปแบบ
- **ค่าใช้จ่าย:** ตอนนี้มีการเรียกใช้ Anthropic API จริง (มีเครดิต $5 อยู่) และ Firebase อยู่บนแผน Blaze (pay-as-you-go มี free tier ก้อนใหญ่ก่อนเริ่มคิดเงิน) — ควรหมั่นเช็คยอดใช้จ่ายเป็นระยะที่ platform.claude.com (Billing) และ Firebase Console (Usage)
- Firestore Security Rules เปิดกว้างชั่วคราว หมดอายุ 8 ส.ค. 2026 — ควรตั้งกฎรัดกุมก่อนเปิดสาธารณะจริงจัง
- ระบบล็อกอิน Admin เป็นการป้องกันเบื้องต้น ยังไม่ใช่ระบบความปลอดภัยระดับ production เต็มรูปแบบ
