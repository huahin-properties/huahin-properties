# huahin.properties — เพิ่มภาษาฝรั่งเศส + อิตาลี

## สิ่งที่เพิ่มในรอบนี้
เพิ่ม **ภาษาฝรั่งเศส (Français)** และ **ภาษาอิตาลี (Italiano)** ครบทุกจุด:
- ตัวเลือกในเมนูเปลี่ยนภาษาทุกหน้า (หน้าแรก, ค้นหา, รายละเอียดทรัพย์ ฯลฯ) — ทดสอบแล้วใช้งานได้จริง
- คำแปล UI ทั้งหมด (เมนู, ปุ่ม, ป้ายกำกับ) ครบทุกข้อความเหมือน 6 ภาษาเดิม
- ระบบ AI แปลอัตโนมัติ (ตอนเพิ่มทรัพย์ด้วย AI Quick Add) ตอนนี้แปลเป็น 7 ภาษาแทน 5 ภาษา (เพิ่ม fr, it)

รวมภาษาทั้งหมดตอนนี้: ไทย, อังกฤษ, รัสเซีย, จีน, เยอรมัน, นอร์เวย์, ฝรั่งเศส, อิตาลี (8 ภาษา)

## ยังไม่ได้แก้ (ค้างจากรอบก่อน)
เรื่องแก้ไข "กรรมสิทธิ์ (Ownership)" ในหน้าตรวจร่างทรัพย์ AI — ถ้าแก้เป็นภาษาไทยตรงนั้น ปัจจุบันยังติดไปทุกภาษาบนเว็บจริง จะแก้ในรอบถัดไป

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด — โดยเฉพาะ `data.js` และ `AI Quick Add.dc.html`)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบเปลี่ยนภาษาเป็นฝรั่งเศส/อิตาลีในหน้าแรก
