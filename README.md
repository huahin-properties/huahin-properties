# huahin.properties — แก้เมนูแอดมินล้นจอ + ช่องรหัสผ่านมองเห็นตัวอักษร

## อัปเดตล่าสุด
หน้า `Admin Login.dc.html` — ช่อง "รหัสผ่าน" ตอนนี้แสดงตัวอักษร/ตัวเลขที่พิมพ์ให้เห็นชัดเจน (ไม่ใช่จุดสีดำแล้ว) เหมือนช่องชื่อผู้ใช้ จะได้ตรวจสอบได้ว่าพิมพ์ถูกหรือไม่ก่อนกด "เข้าสู่ระบบ"

## สิ่งที่แก้ในรอบก่อนหน้า
เมนูแถบบนสีเข้ม (บาร์ดำ) ของหน้าแอดมินดันจอกว้างเกินเมื่อเปิดในมือถือ ทำให้ต้องเลื่อนซ้าย-ขวา แก้แล้วในไฟล์:

- `Admin Dashboard.dc.html`
- `Owners.dc.html` (หน้าผู้ติดต่อ)
- `Property Map.dc.html`
- `Site Content.dc.html`
- `AI Quick Add.dc.html`

ตอนนี้เมนูจะพับขึ้นบรรทัดใหม่แทนการดันจอกว้าง ทั้งบนมือถือและคอมพิวเตอร์ยังแสดงผลเหมือนเดิมทุกจุด

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด — ไม่มีไฟล์ใหม่ที่ต้องเพิ่ม)
2. รอ GitHub Pages deploy เสร็จ แล้ว Hard Refresh (Ctrl+Shift+R หรือปิด-เปิดแอปมือถือใหม่)

## ยังไม่ได้แก้ (ค้างจากรอบก่อน)
