# huahin.properties — ไฟล์เว็บล่าสุด (เพิ่มปุ่มเลือกภาษาบนแถบบนสุดของมือถือ)

## อัปเดตในรอบนี้
ปุ่มเลือกภาษาบนมือถือ (ไอคอนธง) ย้ายจากด้านล่างสุดของเมนูแฮมเบอร์เกอร์ ขึ้นมาอยู่ **ติดกับไอคอนแฮมเบอร์เกอร์ที่แถบบนสุด** ของทุกหน้า (Home, ค้นหา, รายละเอียดทรัพย์, ฝากขาย/เช่า, เกี่ยวกับเรา, ติดต่อ) ผู้ใช้จะเห็นและเลือกภาษาได้ทันทีที่เข้าเว็บ ไม่ต้องกดเปิดเมนูก่อน

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo ของคุณ ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที)
3. Hard Refresh (Ctrl+Shift+R หรือปิด-เปิดแท็บใหม่บนมือถือ) ก่อนทดสอบ

## แก้ไขไปแล้วก่อนหน้านี้ (สรุปสะสม)
- **รูปภาพอัปโหลดไม่โชว์:** พบสาเหตุจริงคือรูป data URL มี `;` ที่ทำให้ระบบตัดข้อความขาด แก้ไขทั่วทั้งเว็บแล้ว ทดสอบอัปโหลด/บันทึก/reload ยืนยันใช้งานได้จริงทุกหน้า (Admin, Home, ค้นหา, รายละเอียดทรัพย์, ทำเนียบผู้ติดต่อ)
- ข้อมูลทรัพย์ทั้งหมดเก็บใน Firebase กลาง ทุกคนที่เข้าเว็บเห็นข้อมูลตรงกัน

## ⚠️ ข้อควรรู้ (เหมือนเดิม)
- Firestore Security Rules เปิดกว้างชั่วคราว หมดอายุ 8 ส.ค. 2026 — ควรตั้งกฎรัดกุมก่อนใช้งานจริง
- Admin Login ยังเป็นระบบจำลอง ควรผูก Firebase Authentication จริงก่อนเปิดสาธารณะ
