# huahin.properties — ไฟล์เว็บล่าสุด (แก้ AI timeout + URL backend ใหม่)

## อัปเดตล่าสุด
- แก้ปัญหา "AI ประมวลผลไม่สำเร็จ: TypeError: terminated" — เพิ่มเวลาให้ตัวกลาง (Cloud Function) ทำงานได้นานขึ้น (จาก 60 วินาที เป็น 5 นาที) รองรับกรณีมีรูปภาพหลายรูปให้ AI วิเคราะห์
- อัปเดต URL ของตัวกลางในโค้ดหน้าเว็บให้ตรงกับ URL ใหม่ที่ได้จากการ deploy ล่าสุด

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบปุ่ม 🤖 "เพิ่มทรัพย์ด่วนด้วย AI" ในแดชบอร์ดอีกครั้ง ควรทำงานได้ราบรื่นแล้ว

## สรุประบบทั้งหมดที่ใช้งานได้แล้ว
- **Firestore** — เก็บข้อมูลทรัพย์/เจ้าของ/ผู้เช่า/รูปภาพ ให้ทุกคนเห็นตรงกันแบบเรียลไทม์
- **Firebase Cloud Function** — ตัวกลางเรียก Claude AI อย่างปลอดภัย (เก็บ API key ไว้ที่ `functions:secrets`) timeout 5 นาที
- **ระบบล็อกอิน Admin** — `132435` / `qewret` (เปลี่ยนได้ที่หน้า Site Content)
- **อัปโหลดรูปภาพ** — เก็บใน Firestore โดยตรง (บีบอัดอัตโนมัติ)
- **AI Quick Add** — อัปโหลดรูป+เอกสาร+เสียงพูด → AI ร่างประกาศ 6 ภาษา → ตรวจสอบ → เผยแพร่

## ⚠️ ข้อควรรู้ก่อนใช้งานจริงเต็มรูปแบบ
- **ค่าใช้จ่าย:** มีการเรียกใช้ Anthropic API จริง (เครดิต $5) และ Firebase อยู่บนแผน Blaze (pay-as-you-go มี free tier ก้อนใหญ่ก่อนเริ่มคิดเงิน) — ควรหมั่นเช็คยอดใช้จ่ายเป็นระยะที่ platform.claude.com (Billing) และ Firebase Console (Usage)
- Firestore Security Rules เปิดกว้างชั่วคราว หมดอายุ 8 ส.ค. 2026 — ควรตั้งกฎรัดกุมก่อนเปิดสาธารณะจริงจัง
- ระบบล็อกอิน Admin เป็นการป้องกันเบื้องต้น ยังไม่ใช่ระบบความปลอดภัยระดับ production เต็มรูปแบบ
