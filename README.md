# huahin.properties — อัปเดตล่าสุด (ค้นหาด้วยคำ/รหัสบ้าน + แสดงรหัสบ้านบนรูป)

## สิ่งที่เพิ่มในรอบนี้
1. **ช่องค้นหาด้วยคำค้น/รหัสบ้าน** — เพิ่มในหน้าแรก (เหนือตัวกรอง ซื้อ/เช่า/ทำเล/ประเภท) และในแผงตัวกรองหน้าค้นหา ค้นได้ทั้งรหัสบ้าน (เช่น HH-101), ชื่อทรัพย์, โซน หรือคำในคำโปรย
2. **รหัสบ้านโชว์บนรูปปก** — ทุกการ์ดทรัพย์ (หน้าแรก/ค้นหา/ทรัพย์ใกล้เคียง) มีรหัสบ้านซ้อนอยู่มุมล่างขวาของรูป แบบบางๆ ไม่กินพื้นที่เพิ่ม
3. **รหัสบ้านโชว์ในหน้ารายละเอียดทรัพย์** — มุมบนขวาของรูปหลัก คู่กับป้ายสถานะ (ประกาศขาย/เช่า)

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)

## ⚠️ ยังไม่ได้ทำในรอบนี้
- แก้ไข "กรรมสิทธิ์" ในหน้าตรวจร่าง AI ให้แปลได้ครบทุกภาษา (ยังติดเป็นภาษาเดียวข้ามภาษาอื่นถ้าแก้ไขตรงนั้น) — จะทำในรอบถัดไป
