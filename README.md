# huahin.properties — ไฟล์เว็บล่าสุด (เพิ่มภาษาเยอรมัน + นอร์เวย์)

## อัปเดตในรอบนี้
เพิ่ม 2 ภาษาใหม่เข้าระบบเลือกภาษาแล้ว:
- 🇩🇪 Deutsch (เยอรมัน)
- 🇳🇴 Norsk (นอร์เวย์)

รวมเป็น 6 ภาษา (ไทย/อังกฤษ/รัสเซีย/จีน/เยอรมัน/นอร์เวย์) แปลครบถ้วนทั้ง:
- เมนู ปุ่ม ฟอร์ม ทุกข้อความ UI ของเว็บไซต์
- ชื่อทรัพย์ คำอธิบายย่อ และคำอธิบายเต็มของทรัพย์ทั้ง 24 รายการ
- ป้ายชื่อพื้นที่ ประเภททรัพย์ สถานะ และคุณสมบัติทรัพย์ทุกหมวด

ทดสอบแล้ว: เลือกภาษาเยอรมันหรือนอร์เวย์จากเมนู แล้วดูหน้าแรก หน้าค้นหา และหน้ารายละเอียดทรัพย์ ยืนยันข้อความและเนื้อหาแปลถูกต้องครบถ้วน

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo ของคุณ ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที)
3. Hard Refresh (Ctrl+Shift+R) ก่อนทดสอบ

## ⚠️ ข้อควรรู้ (เหมือนเดิม)
- Firestore Security Rules เปิดกว้างชั่วคราว หมดอายุ 8 ส.ค. 2026 — ควรตั้งกฎรัดกุมก่อนใช้งานจริง
- Admin Login ยังเป็นระบบจำลอง ควรผูก Firebase Authentication จริงก่อนเปิดสาธารณะ
- ถ้าเพิ่มทรัพย์ใหม่ผ่านหน้า Admin ในอนาคต ชื่อ/คำอธิบายจะบันทึกเป็นภาษาอังกฤษเท่านั้น (ภาษาอื่นต้องแปลเพิ่มเติมเองทีหลังถ้าต้องการ)
