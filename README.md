# huahin.properties — อัปเดตล่าสุด (แคปชั่น Facebook แสดงสถานที่ใกล้เคียงครบถ้วน)

## สิ่งที่แก้ในรอบนี้
ข้อความสำหรับโพสต์ Facebook ตอนนี้จะแสดง**สถานที่ใกล้เคียงครบทุกแห่งเสมอ** ทั้งภาษาอังกฤษและภาษาไทย ในรูปแบบขีด (-) เดียวกับที่แสดงในหน้าเว็บไซต์ (ก่อนหน้านี้ปล่อยให้ AI เขียนอิสระทำให้บางครั้งเหลือแค่ 1-2 แห่ง) ตอนนี้ระบบจะแนบรายการทั้งหมดต่อท้ายให้อัตโนมัติ รับประกันว่าครบเหมือนในเว็บไซต์ทุกครั้ง

## ยังไม่ได้แก้ (รอบถัดไป)
เรื่องแก้ไขข้อมูล "กรรมสิทธิ์ (Ownership)" ในหน้าตรวจร่าง — ถ้าแก้เป็นภาษาไทยตรงนั้น ปัจจุบันจะติดไปทุกภาษาบนเว็บจริง (ช่องอื่นเช่น ชื่อทรัพย์/คำโปรยไม่มีปัญหานี้แล้ว) จุดนี้ต้องแก้ไขอย่างระมัดระวังเพราะเชื่อมกับหลายหน้า จะดำเนินการในรอบถัดไป

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. ทดสอบเพิ่มทรัพย์ใหม่พร้อมพิกัดแผนที่ แล้วดูข้อความ Facebook ที่ได้
