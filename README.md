# huahin.properties — ไฟล์เว็บล่าสุด (มี AI Quick Add + ระบบล็อกอิน Admin)

## อัปเดตในชุดนี้
- **หน้า "เพิ่มทรัพย์ด่วนด้วย AI"** (ใหม่) — อัปโหลดรูป + เอกสารโครงการ (PDF/รูป ได้หลายไฟล์) + พูด/พิมพ์อธิบายทรัพย์ → AI ร่างประกาศเป็นภาษาไทยให้ตรวจสอบ → แก้ไข/พูดเพิ่มเติมได้จนพอใจ → กดเผยแพร่จะแปลครบ 6 ภาษาอัตโนมัติ (ไทย/อังกฤษ/รัสเซีย/จีน/เยอรมัน/นอร์เวย์) และตั้งชื่อไฟล์รูปแบบ SEO ให้อัตโนมัติ เข้าถึงได้จากปุ่ม 🤖 ในหน้า Admin Dashboard
- **ระบบล็อกอิน Admin จริง** — Username/Password ชั่วคราว: `132435` / `qewret` (เปลี่ยนได้ที่หน้า Site Content ในแดชบอร์ด พร้อมอีเมล/เบอร์สำรองกู้คืน)
- แก้บั๊กรูปภาพขาดหายจากการตัดสตริงที่เครื่องหมาย `;` — แก้ครบทุกจุดแล้ว

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo บน GitHub ที่ระดับ root (ทับไฟล์เดิมทั้งหมด)
2. รอ GitHub Pages deploy เสร็จ (1-2 นาที) แล้ว Hard Refresh (Ctrl+Shift+R)
3. เข้า `Admin Login.dc.html` ล็อกอินด้วย `132435` / `qewret`
4. ทดสอบฟีเจอร์ "เพิ่มทรัพย์ด่วนด้วย AI" จากปุ่ม 🤖 ในแดชบอร์ด — **ทดสอบบนเว็บจริงหลัง deploy เท่านั้น** ปุ่มไมค์จะไม่ทำงานถ้าเปิดผ่านหน้าต่างพรีวิวที่ซ้อน iframe (เช่นในนี้)

## ⚠️ ข้อควรรู้ก่อนใช้งานจริง
- Firestore Security Rules เปิดกว้างชั่วคราว หมดอายุ 8 ส.ค. 2026 — ควรตั้งกฎรัดกุมก่อนเปิดสาธารณะจริงจัง
- ระบบล็อกอิน Admin นี้เป็นการป้องกันเบื้องต้น (เก็บรหัสผ่านใน Firestore) ยังไม่ใช่ระบบความปลอดภัยระดับ production เต็มรูปแบบ — แนะนำเปลี่ยนเป็น Firebase Authentication จริงก่อนใช้งานกับข้อมูลลูกค้าจริงจำนวนมาก
- ฟีเจอร์ AI ใช้ `window.claude.complete` ซึ่งทำงานได้เฉพาะภายในสภาพแวดล้อมของ claude.ai เท่านั้น — ถ้านำโค้ดไปรันบนโดเมนจริงนอกระบบนี้ (เช่น GitHub Pages ตรงๆ) ฟังก์ชัน AI ร่างประกาศจะไม่ทำงาน ต้องเชื่อมต่อ Claude API จริงแยกต่างหาก (มี key และ backendของคุณเอง) จึงจะใช้ฟีเจอร์นี้ได้บนเว็บจริงนอกระบบนี้
